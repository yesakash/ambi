# ML Training Platform - MVP

This is the Minimum Viable Product for an integrated platform to train different machine learning models.

## Features (MVP)
- Upload a CSV dataset.
- Specify the target variable for training.
- Select from a list of standard classification models (Logistic Regression, Random Forest, SVM).
- Train the selected models on the dataset.
- View a results leaderboard, ranked by model accuracy.
- Download the trained model file (`.pkl`).
